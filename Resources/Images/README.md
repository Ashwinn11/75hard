# Image drop-in guide

Drop `.jpg` files in this folder using these **exact names** (lowercase, case-sensitive).
This folder is bundled as a **folder reference**, so you can add/replace images and just
**rebuild in Xcode** — no `xcodegen generate` needed. Every image is optional: anything missing
falls back to a clean on-brand gradient, so the app always looks intentional.

## Mission card photos (portrait ~3:4)
`card_move` · `card_water` · `card_read` · `card_eat` · `card_alcohol` · `card_photo`

## Onboarding welcome wall (square-ish, 12)
`onb_g1` … `onb_g12` — editorial lifestyle (fitness, smoothie, yoga, salad, skincare, books,
running, pilates, flowers, journaling, sunrise, portrait). Any subset works.
